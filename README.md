# 4Us App Landing Page

A modern, responsive landing page for the 4Us mobile app built with Next.js, React, and Tailwind CSS.

## Features

- 🎨 Modern UI with Aceternity UI grid and dot background
- 📱 Fully responsive design optimized for mobile devices
- 🎯 App Store and Google Play download buttons
- ✨ Rewards profile activation form
- 🌈 Gradient text effects and modern styling
- ⚡ Fast and optimized with Next.js

## Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Aceternity UI** - Grid background component

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Asim1921/4us.git
cd 4us
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
